# ra8875-stm32

Minimal RA8875 display driver for STM32 HAL over SPI.

## Current scope

- STM32 HAL
- 4-wire SPI
- basic display initialization
- early drawing support
- example STM32CubeIDE project

## Tested hardware

- STM32F767ZI
- RA8875-based TFT display board
- 3.3V logic

## Repository structure

- `driver/` - RA8875 driver source
- `examples/display_test/` - example STM32CubeIDE project

## Status

Work in progress.
